# 🤖 AI Prompt Generator

一个简单好用的网页工具，帮助你快速生成高质量的 AI 提示词（支持 ChatGPT、Grok、Claude 等）

## ✨ 功能
- 输入主题或任务 → 一键生成优化提示词
- 支持多种场景（写作、编程、翻译、脑暴等）
- 一键复制提示词
- 历史记录（今天生成的提示）

## 🚀 使用方法
直接打开 `index.html` 在浏览器中使用，无需安装！
